## 🖥️ UBB-OS – Operating Systems

Assignments, projects and tests from the *Operating Systems* course at Babeș-Bolyai University (UBB), covering core OS concepts and system-level programming.

Includes:
- 💻 Shell scripting with `grep`, `sed`, `awk`, and other tools
- 🔗 Pipe and FIFO (named pipes) communication
- 🧑‍💻 Multi-threading and parallel processing
- 🔐 Semaphores and synchronization techniques
- 🔄 Process control with `fork` and `wait` system calls

A collection of exercises and implementations focused on understanding the core concepts of operating systems, system calls, process management, and inter-process communication.
